# Pulxio Maternity Installer

Official desktop installer for Pulxio Maternity fetal monitoring system.

Download the latest release from the [Releases](https://github.com/pulxio/pulxio-installer/releases) page.

## System Requirements

- Windows 10/11 (x64)
- 8 GB RAM minimum
- 2 GB free disk space

## What's Included

The installer bundles all required services:

- Pulxio Maternity desktop application
- PostgreSQL database
- InfluxDB time-series store
- Mosquitto MQTT broker
- FetalAI monitoring engine
